# HostelHive App

Hostel Hive - Flutter application for hostel management.

## Repository
- GitHub: https://github.com/akashfdev-gif/hostelhive_app

## Setup
1. Clone: `git clone https://github.com/akashfdev-gif/hostelhive_app.git`
2. Flutter: `flutter pub get`
3. Run: `flutter run`

## GitHub Integration
- Remote origin set.
- Authenticated via GitHub CLI (gh).
- Push changes: `git push origin main`

Built with Flutter 3.x.
